# JavaScript Interview Questions #2 - Variables

** Question 1: **

How can we declare a variable in JavaScript?

** Answer: **

We can declare variables in JavaScript using `var`, `let` and `const` keywords.

```javascript
var a;
let b;
const c = 3;
```
