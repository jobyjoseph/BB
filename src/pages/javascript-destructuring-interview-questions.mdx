# JavaScript Destructuring Interview Questions

**Question**

What is destructuring in JavaScript?

**Solution**

Destructuring in JavaScript enables breaking down a larger data structure and assign the individual elements to variables or properties. It is a feature that came as part of ES6.

**Question**

Here is a JavaScript code snippet.

```javascript
var name = obj.name;
var age = obj.age !== undefined ? obj.age : 0;
```

Refactor above code using JavaScript destructuring.

**Solution**

```javascript
var {name, age = 0} = obj;
```

**Question**

We have an array and elements from the array are assigned to variables.

```javascript
var arr = ["B", "a", "c", "k"];
var first = arr[0];
var second = arr[1];
```

Refactor this code using destructuring.

**Solution**

```javascript
var arr = ["B", "a", "c", "k"];
var [first, second] = arr;
```
