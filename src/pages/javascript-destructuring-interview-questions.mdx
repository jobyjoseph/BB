# JavaScript Destructuring Interview Questions

**Question**

What is destructuring in JavaScript?

**Solution**

Destructuring in JavaScript enables breaking down a larger data structure and assign the individual elements to variables or properties. It is a feature that came as part of ES6.

**Question**

Here is a JavaScript code snippet.

```javascript
var name = obj.name;
var age = obj.age !== undefined ? obj.age : 0;
```

Refactor above code using JavaScript destructuring.

**Solution**

```javascript
var {name, age = 0} = obj;
```

**Question**

We have an array and elements from the array are assigned to variables.

```javascript
var arr = ["B", "a", "c", "k"];
var first = arr[0];
var second = arr[1];
```

Refactor this code using destructuring.

**Solution**

```javascript
var arr = ["B", "a", "c", "k"];
var [
  first,
  second
] = arr;
```

**Question**

What is the output of following code snippet?

```javascript
var arr = ["B",, "c", "k"];
var [first, second] = arr;

console.log(second);
```

**Solution**

The output printed is `undefined`. Here the second element of the array holds `undefined` and that is assigned to `second`.

**Question**

What is the output of following code snippet?

```javascript
var arr = ["B", null, "c", "k"];
var [first, second] = arr;

console.log(second);
```

**Solution**

The output is `null`. `null` is not strictly equal to `undefined`.

**Question**

What is the output of following code snippet?

```javascript
var arr = ["B",, "c", "k"];
var [first, second = 10 ] = arr;

console.log(second);
```

**Solution**

It prints `10`. Since `second` evaluates to `undefined`, JavaScript assigns the default value of `10`.

**Question**

What is the output of following code snippet?

```javascript
var arr = ["B", null, "c", "k"];
var [first, second = 10 ] = arr;

console.log(second);
```

**Solution**

It prints `null`. `second` will take the default value of 10 only if it evaluates to `undefined`, not any other falsy values.

**Question**

How can we refactor below code using destructuring?

```javascript
var arr = ["B", "a", "c", "k", "b", "e", "n", "c", "h", "e", "r"];
var first = arr[0];
var second = arr[1];
var rest = arr.slice(2);
```

**Solution**

```javascript
var arr = ["B", "a", "c", "k", "b", "e", "n", "c", "h", "e", "r"];
var [
  first,
  second,
  ...rest
] = arr;
```

**Question**

What is the output of following code snippet?

```javascript
var arr = ["B", "a"];
var [
  first,
  second,
  ...rest
] = arr;

console.log(rest);
```

**Solution**

It prints `[]`. Note: it does not print `undefined`.

**Question**

Is this code valid?

```javascript
var arr = ["B", "a", "c", "k"];
var first, second, third;
[
  first,
  second,
  third
] = arr;
```

**Solution**

Yes. It is valid. Destructuring is all about assignment. So we can declare the variables first separately and then assign it using destructuring.

**Question**

Here is a JavaScript code snippet.

```javascript
var arr = ["Joe", 23, "Male"];
var profile = {};
profile.name = arr[0];
profile.age = arr[1];
profile.gender = arr[2];
console.log(profile);
```

Refactor above code using destructuring.

**Solution**

```javascript
var arr = ["Joe", 23, "Male"];
var profile = {};
[
  profile.name,
  profile.age,
  profile.gender
] = arr;
console.log(profile);
```

**Question**

Here we have a code that copies first two elements of an array to another array.

```javascript
var arr = ["Joe", 23, "Male"];
var newArr = [];
newArr[0] = arr[0];
newArr[1] = arr[1];
console.log(newArr);
```

Refactor it using destructuring.

**Solution**

```javascript
var arr = ["Joe", 23, "Male"];
var newArr = [];
[
  newArr[0],
  newArr[1]
] = arr;
console.log(newArr);
```

**Question**

Here is an array.

```javascript
var arr = ["B", "a", "c", "k"];
```

How to pick only `"B"` and `"k"` from `arr` using destructuring?

**Solution**

```javascript
var arr = ["B", "a", "c", "k"];
var [
  first,
  ,
  ,
  fourth
] = arr;
```

We can skip elements by simply putting comma separator.

**Question**

Here is a JavaScript code snippet to swap two numbers.

```javascript
var a = 2;
var b = 5;
var temp = a;
a = b;
b = temp;
console.log(a);
console.log(b);
```

Refactor the code without using `temp` variable.

**Solution**

```javascript
var a = 2;
var b = 5;
[b, a] = [a, b]
console.log(a);
console.log(b);
```
